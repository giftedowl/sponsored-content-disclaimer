Add a custom disclaimer message to any blog post. The message can be displayed either above or below the content. Add custom CSS to provide a consistent style across your blog.

== Description ==

If you frequently write sponsored content or affiliate reviews, Sponsored Content Disclaimer allows you to add a custom disclaimer message to each post. Styling of the disclaimer message can be changed across the entire site.

Features:

*  Add a custom message above or below a post
*  Custom message will not be displayed in the excerpt
*  Change styling of all message boxes

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/sponsored-content-disclaimer` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Appearance->Sponsored Content screen to configure the styling
1. Add a custom message while editing a post


== Frequently Asked Questions ==

= How do I add a disclaimer message to a post? =

Add or edit a blog post. You will see a meta box labeled Sponsored Content Disclaimer below the post editing area. Add a custom message. You can select to display the message above or below the post content.

= How do I modify the styling of the Sponsored Content Disclaimer box =

The sponsored content disclaimer message is wrapped in a div container. The container has a class labeled scd-message-box.

Go to Appearance->Sponsored Content to add any custom CSS you want.
